Página web de boda. Publicada en Netlify, editada desde GitHub.

---

## 🔧 Cómo abrir o cerrar la página al público

1. Abre el archivo **`index.html`** (clic en el archivo → ícono del lápiz ✏️)
2. Busca esta línea:
   ```
   var SITE_OPEN = false;
   ```
3. Cámbiala a:
   ```
   var SITE_OPEN = true;
   ```
4. Haz clic en **"Commit changes"** (botón verde arriba a la derecha)
5. Netlify publica el cambio automáticamente en ~30 segundos ✅

## ✏️ Cómo editar textos

Para cambiar cualquier texto (nombres, direcciones, fechas, el link de Luma):

1. Haz clic en el archivo que quieres editar (ej: `wedding.html`)
2. Presiona el ícono del **lápiz ✏️** arriba a la derecha
3. Edita el texto directamente
4. Haz clic en **"Commit changes"** → cambio publicado en segundos
